A Version Control System (VCS) is a tool used in software development and collaborative projects to track and manage changes to source code.
It enables:
- Tracking changes
- Restore old versions
- Collaboration
- Branching
- Logging 

## Important terms
|Term|Meaning|
|---|---|
|Repository (repo)|Project tracked by VCS|
|Commit|Saved snapshot|
|Branch|Separate line of development|
|Merge|Combine changes|
|Clone|Copy repository|
|Pull|Download latest changes|
|Push|Upload your changes|

## Types of version control system

1. Local Version Control System
2. Centralized Version Control System
3. Distributed Version Control System
